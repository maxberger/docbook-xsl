Installation instructions

Procedure 1. To install the package

  1. Confirm that you have Java 1.5 or higher and Ant 1.6.5 or higher installed.

  2. Download the package: docbook-webhelp.zip and unzip it somewhere convenient.

  3. Edit xsl/driver.xsl so that it imports chunk.xsl from the DocBook XSLs from your local drive: 

    <xsl:import href="http://docbook.sourceforge.net/release/xsl/current/xhtml/chunk.xsl"/>

  4. Test the package by running ant doc. It should generate a copy of this documentation in the doc directory.

  5. If you wish to use the included build.xml file for your own content, create a build.xml file that imports the one from this package: 

    <project>
  <property name="input-xml" value="path-to/yourfile.xml"/>
  <property name="frameset.base.dir" value="path-to/output-dir"/>
  <property name="xslt-processor" value="c:/workhead/export/saxon/6.5/lib/saxon.jar"/>
  <property name="docbookx.dtd value="file:///c:/workhead/export/DocBookDTD/4.5/docbookx.dtd"/>
  <import file="path-to/docbook-webhelp/build.xml"/>
</project>

  6. From the directory containing your newly created build.xml file, type ant webhelp to build your document.

  7. Add some code to the main ant script to copy your images to the output directory.

Note

By design, the indexer does not index index.html or ix01.html. Unfortunately, this is currently hard coded in the indexer.
