Cascading Stylesheet for DocBook XML

About

DocBook is an XML dialect for writing documentation.

The Cascading Stylesheet provided here allows you to directly view a styled XML document in software that supports XML styled with CSS2 (e.g. a recent Mozilla, Opera or Internet Explorer)

Status

Works sometimes. Not extensively tested.

Usage

To use, alter your XML file to include a stylesheet directive,

<?xml version="1.0"?>
<?xml-stylesheet href="docbook-css-0.2/driver.css" type="text/css"?>

<book>
...
